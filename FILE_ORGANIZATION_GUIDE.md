## WORKFLOW: Adding New Files to Enigma

### Step 1: Add Files to Appropriate Folder
```bash
# Example: Adding a PDF book to Skills
cp ~/Downloads/product_book.pdf ~/Enigma/01_Skills_And_Education/Product_Books/

# Example: Adding Product Genome chapter
cp ~/Documents/chapter_draft.docx ~/Enigma/03_Projects/Product_Genome/
```

### Step 2: Re-index Documents (Makes Them Searchable)
```bash
cd ~/Enigma
python3 scripts/index_documents.py
```

This updates the semantic search index so Enigma can find and cite your new documents.

### Step 3: Query or Edit
```bash
# Query the new content
python3 scripts/smart_query.py "What does the product book say about..."

# Or edit a Word doc (coming soon)
python3 scripts/edit_document.py --file path/to/document.docx
```
